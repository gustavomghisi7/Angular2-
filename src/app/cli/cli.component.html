Angular command-line interface (CLI)

Criando e Rodando o Projeto

Opção 1:

ng new NomeProjeto

cd NomeProjeto

ng serve

Opção 2:

mkdir NomeProjeto

cd NomeProjeto

ng init

ng serve

Criando um Componente ou Serviço

ng g c NomeComponente

ng g component NomeComponente

ng g s NomeServiço

ng g service NomeServiço

Scanner do código e Testes unitários

ng lint

Scanner no código e procura por pequenos erros

ng test

Rodar o testes unitários do Jasmine
